LamentXU@blog:~/about$ cat ./php_core_developer

php 语言是世界闻名的服务端编程语言。它由一个 core 和 几十个 ext 组成。这个 ext 类似于 python 的标准库，属于 core 的工具箱供人们使用。

我维护着 php 语言的 intl 这个 ext。这个 ext 名是 international 的缩写。即，为国际化提供支持。

你不一定知道 intl 这个库是做什么的。但是，只要你以 php 语言开发网页，你一定用过我维护的代码（除非你的网页只有 ASCII 字符）。只要实现 i18n 的功能，你必然绕不开利用 intl。（事实上 intl 安装在任何一个 php 发行版上（>= 5.3）。离开它你的网页很难工作。）

说起来，我十分享受维护这样一个语言中必不可少的部分。我对 php 语言的所有贡献，可以在 Weilin Du 的 php 开发日志里找到

intl 这个 ext 从 2008 年诞生开始已经有很长的历史了，也一直默默地支撑着无数 php 网页的运行。严格上来说，我维护的文件大概是：
