HKCERT 2025 final 感想

LamentXU
2026-02-08 (Updated: 2026-02-08)
CTF
CTF, Pwn, Web Security

于是我终于决定缺席了第三次模拟考试，同我唯一的一位老队友一起踏上了追逐梦想的舞台。在这里，由两个高中生组成的队伍，最终在以四个大学生组成的队伍中在香港最大的网络安全竞赛中夺得亚军。

从此，我的高中，就再没有了遗憾。我可以，也本就应该，有昂首阔步的信念，迎接高考，以及成年的自己。

遗憾贯穿人生始终，也正因如此，使那几次本就为数不多的成功显得熠熠生辉。故事的结尾，如同故事的开端一般，由我们开启，也由我们结束。

竞赛

初赛

我和 jerrythepro 最初打算报名这场比赛时本就是抱着玩一玩的心态来的。毕竟同为高中生，彼时的我们有更重的压力和任务在身。同时，虽然我们都是香港户籍，但 jerrythepro 并不在香港读书，我们也因此报名不了属于中学生的赛道，也只能在公开组同大学生们竞争。

然而初赛，我们还是相约至少打好第一波题目。我记得，那是我第一次逃课出去打 CTF，也就慌慌张张地打出了一份答卷。

Jerry 呢，与往常一样，在香港的公开组中豪取 3 个 pwn 一血。可以说，我们的第一波打的远比想象中容易。（队名：EICS1）

然而后面的题目我们由于要上学，就只能且做且珍惜了。最后，我们以第四名的成绩勉强进入决赛轮。

两天后呢，Jerry 已经得到大学的通知书了。我们于是出去吃了顿饭庆祝，也提到了决赛的事。

再三思考过后，我们还是决定参加公开组的决赛。

决赛

决赛前一天，我的港澳台通行证消失了。真的消失了。我当时在深圳湾口岸凭着 2025 年 11 月 25 日试点的人脸识别的方式过的关。真是虚惊一场。

在酒店是与我所在的国际联合战队 W&M 的队员拼房的（l4n, q1uf3ng, inkey）jerry 也来了。备赛了一晚上，我们一起去附近的维多利亚港散步。

我已经忘记上次这么开心是什么时候了。

何夜无月，何处无竹柏？真的，这是我到现在所能欣赏的，最漂亮的 Victora Harbour。

第二天，坐上决赛的专巴，比赛开始。但是，在第 0 轮开始之际，我和 jerry 的网络还没有配置完成，我们就这样在比赛的前 20 分钟没法打输出，也就迎来了最差开局——倒数第一。

终于我们连上了内网（由于网络路由优先级很诡异，我处于一种内网与外网二者不可得兼的状态。。）反击开始。很快我找出了 babypython 一题打全场的漏洞一下子刷到了第三名，随后 Jerry 也开始发力，凭借着又一个 pwn 的打全场的漏洞，我们在中午成功将比赛逆转，成为了 rank 1。

然而下午，我的情况就开始糟糕了。JAVA 一向是我的弱项，下午新上的 JAVA 的一个最基本的洞是 Lua 沙箱中的不出网的 curl 参数注入。然而我最终还是没做出来，甚至没 patch 好。或者说，我根本不会 patch。

由于第一次打 AWD，我根本不知道如何分析 checker 的行为。运维删马的时候也手忙脚乱的。可以说，我的防守做的简直是非常差。我们本因此输掉比赛。但偏偏我们的攻击又很好——三个 web 题，我有两个 web 一血。Jerry 的 pwn，更是能稳定每轮出全场 7 个 flag。即使只有两个人，防守人数严重不足的我们居然还能靠攻击分弥补防守分…… 最后我们干脆直接 down 掉了所有 web 服务，只靠攻击来弥补每轮扣的 150 分。这样，我们拿到了亚军。

W&M，当然，毫无疑问的国际组冠军，即使他们只有三个人。

就这样，我们创造了奇迹。

以防你不知道，上图中五个人有三个成年人是国际组冠军，有两个未成年人是公开组亚军。

赛后

赛后我们一起去吃了 HK 最正宗的海鲜大排档。太好吃了。（其实是我这个 native 带错了路最后歪打正着的hh）

总结

我依稀记得两年前，是 Jerry 来找我到时候，我才知道 CTF 这个东西。那天高一的我绝对不会想到两年后，站在赛场上的依然只有我们，但只有我们，也就够了。

这是我们高中的最后一场比赛。从此，我的老队友大概要去美国读大学咯（唉）。比赛结束后，我说，这是我们的最后一场比赛。他说，defcon 再见。好。

V&NCTF 结束后我就一直在想，AI 如此发达的时代下还要我们做什么呢？我如今找到了答案。

我要看鹰击长空，鱼翔潜底；我要问苍茫大地，谁主沉浮。当我下次踏上 CTF 的赛场时，必不能再身穿校服，也不会再有买桂花同载酒的少年的心气。然而，我已经没有遗憾。即使亚军，我们以如此劣势踏上赛场的一刻，我就已经感受到成长。

鲁莽和怯懦的折中是勇气。但如果硬要我选一个，我会选鲁莽。因为鲁莽，比怯懦更接近勇敢。
