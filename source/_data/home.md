## php 内核维护者

php 语言是世界闻名的服务端变成语言。它由一个 core 和 几十个 ext 组成。这个 ext 类似于 python 的标准库，属于 core 的工具箱供人们使用。

[我维护着 php 语言的 intl 库](https://github.com/php/php-src/blob/master/EXTENSIONS#L354)。这个库名是 international 的缩写。即，为世界各地的人提供支持。

你一定不知道 intl 这个库是做什么的。但是，只要你以 php 语言开发网页，你说不定用过我维护的代码。只要实现 i18n 的功能，你必然绕不开利用 intl。

说起来，我十分享受维护这样一个语言中必不可少的部分。我对 php 语言的所有贡献，可以在 [Github commit history](https://github.com/php/php-src/commits?author=LamentXU123) 里找到。
