# 哈希表 HashTable

python 中没有所谓的 HashTable 类型所以需要自定义。HashTable 对 python 这种高级语言来说更像是一种抽象的思想。

其实，在本人看来 C 中的 HashTable 就是 python 中 Dict 类型的子集。可能有误。

## 两数之和 - 简单

```
给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target  的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。
```

此处的哈希表实现很明显，是以索引为 value，值为 key，然后遍历值即可在 O(n) 时间复杂度下解决。此例中我们可以动态地向表中添加值。最坏条件下，在最后一次遍历中必然能求解。

```
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hash_map = {} 
        for i, num in enumerate(nums):
            complement = target - num
            if complement in hash_map:
                return [hash_map[complement], i]
            hash_map[num] = i
```

## 字母异位词分组 - 中等

```
给你一个字符串数组，请你将字母异位词组合在一起。可以按任意顺序返回结果列表。
（字母异位词是通过重新排列不同单词或短语的字母而形成的单词或短语，并使用所有原字母一次） 
```

这里最一开始想到的是元组 set 类型作为哈希表的 key。因为元组具有无序性，这就很适合作为字母异位词的 Hash。但是，由于元组具有唯一性。对于含有重复字母的单词无法正确判断。

因此这里采用 sorted() 方法求字符串 Hash。

```
from collections import defaultdict
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        hashmap = defaultdict(list)
        for i in strs:
            hashmap[''.join(sorted(i))].append(i)
        return list(hashmap.values())
```

注：defaultdict 在 python 的 HashTable 的用例中是极其有用的。

## 最长连续序列 - 中等

```
给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 O(n) 的算法解决此问题。
```

首先对于此题而言，给定列表中重复的数字毫无意义。因此我们可以利用 set 的方式去除重复数字达成小优化。

```
from typing import List

class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        if not nums:
            return 0
        
        num_set = set(nums)
        max_len = 0
        
        for num in num_set:
            if num - 1 not in num_set:
                current_num = num
                current_len = 1
                while current_num + 1 in num_set:
                    current_num += 1
                    current_len += 1
                
                max_len = max(max_len, current_len)
        
        return max_len
```
