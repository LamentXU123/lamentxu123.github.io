php 内核维护者

php 语言是世界闻名的服务端编程语言。它由一个 core 和 几十个 ext 组成。这个 ext 类似于 python 的标准库，属于 core 的工具箱供人们使用。

我维护着 php 语言的 intl 库。这个库名是 international 的缩写。即，为世界各地的人提供支持。

你不一定知道 intl 这个库是做什么的。但是，只要你以 php 语言开发网页，你一定用过我维护的代码（除非你的网页只有 ASCII 字符）。只要实现 i18n 的功能，你必然绕不开利用 intl。
